# Map Equivalent Purpose Framework

This is the repository for the Map Equivalent Purpose (MEP) Framework.
This repository is meant to be a living and evolving version of the MEP Framework. Please open issues or pull requests with ideas, thoughts, updates, or contributions to the framework.

# Objective

The objective of the MEP Framework is to be a systematic evaluation tool for determining equivalent purpose of a text map compared with a visual map. The purpose of every map, regardless of any other considerations, is to communicate generalized spatial information and relationships. If this is not the objective of the map, then a map should not be used, and another representation (such as a table) should be used instead.

# Publication
The initial publication of this framework is at:


Biggs, B., Sloan, D., Oppegaard, B., Giudice, N. A., Coughlan, J. M., & Walker, B. N. (2026). Systematically Evaluating Equivalent Purpose for Digital Maps. Journal on Technology and Persons with Disabilities, 13. Preprint: https://arxiv.org/abs/2512.05310
